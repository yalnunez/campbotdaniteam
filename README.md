
# CAMP Bot - Dani Team

Tampermonkey automation bot that monitors agent state time in CAMP, executes autoclick to change states, and sends notifications to associates. Reduces problematic state time by 63.2%, recovering 12.8 productive hours per week for the team. Developed by Yalnunez.

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) in your browser
2. Click on the `.user.js` file in this repository
3. Click "Raw" and Tampermonkey will prompt you to install

## Auto-Update

This script updates automatically via Tampermonkey. Make sure "Check for updates" is enabled in your Tampermonkey settings.
